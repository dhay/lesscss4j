What is LessCSS4j?
------------------

LessCSS4j is an implementation of the LESS language defined by the Ruby Gem
(http://lesscss.org).  It is designed to be a library for use in your own
applications or as a stand alone command line application.  In both cases, the
goal is to "compile" a LESS file into valid CSS.
